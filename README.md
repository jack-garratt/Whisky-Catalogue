# Whisky-Catalogue

A cataloging system to catalogue a large whisky collection reasonably quickly. Storing all data about a bottle in a CSV file as well as front and rear photos in a permanent storage location on the users device.

## Features
- View live camera feed
- Take images of front and rear using keyboard shortcuts
- Enter details about bottle
- Save images to folder
- Save bottle details to CSV

## Requirements
- Python(tested) 3.13.7+
- [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
- [Pillow](https://github.com/python-pillow/Pillow)
- [CV2](https://github.com/opencv/opencv)

Install dependencies:
```bash
pip install -r requirements.txt
```
Run Program
```bash
python camera_controller.py
python main.py
```
